Mentorship Drill Bot

Automated drill tracking & reminder system for mentees in mentorship group

v1.0 · Node.js · Baileys
⚠ Problem 📉

Fewer mentees were sending drill messages in the group, making it hard to track participation manually.

✓ Solution 🤖

Built an automated bot that monitors the group, records drill submissions, and reminds non-compliant mentees daily.

Bot Functions
01
Group Monitoring
Listens exclusively to the configured WhatsApp group and ignores all other chats.
02
Keyword Detection
Flags a submission when a message contains "prayer", "study", or "quiet time" — regardless of format.
03
Drill Recording
Timestamps each mentee's last submission in a local JSON file for audit and tracking.
04
Daily Reminders
Every day at 10:00 AM the bot scans for mentees inactive for more than 1 day and sends them a private reminder.
05
Smart Name Resolution
Uses saved phone contact names first, then WhatsApp display names, so reminders are always personalised.
06
Activity Logging
All drill completions and sent reminders are logged with timestamps for mentor review.
07
Auto Reconnect
If the WhatsApp connection drops, the bot automatically reconnects within 5 seconds without manual intervention.
08
New Mentee Detection
Automatically registers unknown senders as new mentees so no one slips through the cracks.
09
Web Dashboard
Live browser view of mentees, drill history, and reminder logs — accessible from any device.

⚠ Known Limitations

No database — data is stored in local JSON files and may be lost on server redeploy.
Hosted on free tier — subject to monthly hour limits and no persistent disk storage.
Cannot track individual drill tasks per mentee (e.g. Prayer ✓, Study ✗, Quiet Time ✓).
Requires re-scanning QR code after each redeploy due to ephemeral filesystem.
